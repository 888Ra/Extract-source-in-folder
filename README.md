# Extract-source-in-folder
提取文件夹中的源代码保存到word文件中


#### 介绍
通过python脚本来提取文件夹中的源代码，包括  '.java' '.py', '.java', '.c', '.cpp', '.js', '.html', '.css', '.rb', '.go','.vue'等类型的源代码来方便的重复操作
